## Challenge Literatura

Esta aplicación tiene como objetivo aplicar los conocimientos sobre Spring y el manejo de API, utilizando la API de Gutendex.

### Requisitos Previos

- Java JDK 22 o superior
- Maven 4 o superior
- PostgreSQL 16 o superior (debe estar instalado y configurado con la base de datos adecuada)
- Acceso a la API Gutendex

### Configuración

Antes de ejecutar la aplicación, asegúrate de configurar correctamente los siguientes aspectos:

1. **Configuración de PostgreSQL:**
   - Asegúrate de tener una base de datos PostgreSQL configurada y accesible.
   - Verifica que las credenciales de conexión estén actualizadas en el archivo de configuración de la aplicación (si es necesario).

2. **Configuración de la API Gutendex:**
   - Asegúrate de tener acceso a la API Gutendex.
   - Verifica que la URL de la API Gutendex esté correctamente configurada en el archivo de la aplicación.

### Uso

La aplicación proporciona las siguientes funcionalidades:

1. **Libros Disponibles:** Permite obtener la cantidad de libros disponibles en la API Gutendex.
2. **Buscar libros por título:** Busca libros por su título ingresado por el usuario.
3. **Buscar libros por autor:** Busca libros por el nombre del autor ingresado por el usuario.
4. **Buscar libros por idioma:** Permite buscar libros en un idioma específico.
5. **Buscar autores vivos en un año determinado:** Busca y muestra autores que estuvieron vivos en un año específico.
6. **Listar libros almacenados:** Muestra la lista de libros almacenados en la base de datos PostgreSQL.
7. **Listar autores almacenados:** Muestra la lista de autores almacenados en la base de datos PostgreSQL.
8. **Almacenar libro por ID:** Permite almacenar un libro específico utilizando su ID en la base de datos PostgreSQL.

Sigue las instrucciones en pantalla para navegar por las diferentes funcionalidades de la aplicación.
